🌟 Maze Explorer 3D 🌟

Grab the coins. Find the door. Beat every level!

📘 How to Play

  • 🖱️ Move the mouse to look around (click the game first).
  • 🕹️ Walk with the Arrow keys or W A S D.
  • ⬆️ Press Space to jump over the orange hurdles.
  • 🪙 Grab every coin to unlock the exit door.
  • 🚪 Walk into the green door to reach the next level.
  • 👻 Run from the ghosts — if one touches you, game over!

📜 Rules

  • Collect all coins before the door opens.
  • If a ghost touches you, the game is over.
  • Each level is bigger, with more (and faster) ghosts.
  • Use the little map in the corner if you get lost.
  • Finish all 5 levels to win the game. Have fun! 🎉
